h1. User environment files

h2. bash_profile

Contains alias for common rails, git commands and some more.

h2. irbrc

Various features to make the ruby console more friendly.

h2. Installation

Rename each <tt>file</tt> to <tt>.file</tt> and place it under your home directory, possibly appending it to your existing <tt>.file</tt>.
